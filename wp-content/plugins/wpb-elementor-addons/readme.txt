== Description ==

<a href="https://wordpress.org/plugins/elementor/" rel="nofollow">Elementor</a> is the number one page builder for WordPress currently available now in the market. It has a super easy user interface, anyone can design complex web pages without knowing any code.

WPB Elementor Addons can take your site to the next level by adding a number of premium quality addons with a huge number of customization options. It's one of the Best Elementor Addons pack. All the addons are a premium quality modern design.

You can build any professional websites with this addon pack. We have elements for any type of business, corporate or personal websites. The team, services, pricing table addon allow you to integrate it for any type of business websites. 

This addons pack works with any standard free or premium WordPress themes. No matter you are using a free theme, our Best Elementor Addons will always make you feel like a premium theme.

### Available Addons

* **<a href="https://demo.elementorkit.com/post-grid-or-slider/" target="_blank" title="Post Grid and Slider">Post Grid/Slider</a>** that displays any post types posts in a nice grid or slider.
* **<a href="https://demo.elementorkit.com/pricing-table/" target="_blank" title="Pricing Table">Pricing Table</a>** to increases your sales.
* **<a href="https://demo.elementorkit.com/testimonial/" target="_blank" title="Testimonials Grid and Slider">Testimonials Grid/Slider</a>** can show your clients feedbacks.
* **<a href="https://demo.elementorkit.com/team-members/" target="_blank" title="Team Members Grid and Slider">Team Members Grid/Slider</a>** for showing all of your team members.
* **<a href="https://demo.elementorkit.com/service-box/" target="_blank" title="Service Box">Service Box</a>** for showing your offer for your customers.
* **<a href="https://demo.elementorkit.com/fancy-list/" target="_blank" title="Fancy List">Fancy List</a>** can show your list items in a beautiful way.
* **<a href="https://demo.elementorkit.com/content-box/" target="_blank" title="Content Box Grid and Slider">Content Box Grid/Slider</a>** can present any static content with an image.
* **<a href="https://demo.elementorkit.com/logo-slider/" target="_blank" title="Logo Slider">Logo Slider</a>** can showcase your sponsors/featured logos in a slider.
* **<a href="https://demo.elementorkit.com/counterup/" target="_blank" title="Counter Up">Counter Up</a>** can show your statics in a nice counter.
* **<a href="https://demo.elementorkit.com/video-popup/" target="_blank" title="Video Popup">Video Popup</a>** that displays a video in a popup, can be used for video call to action.
* **<a href="https://demo.elementorkit.com/filterable-gallery/" target="_blank" title="Filterable Image Gallery">Filterable Image Gallery</a>** to showcase your images with filtering options.
* **<a href="https://demo.elementorkit.com/content-timeline/" target="_blank" title="Content Timeline">Content Timeline</a>** to show any type of content in a timeline. Support for image, iframe, ShortCode, icon, date etc.
* **<a href="https://demo.elementorkit.com/slider/" target="_blank" title="Slider">Slider</a>** can show a slideshow of images, contents and buttons.
* **<a href="https://demo.elementorkit.com/news-ticker/" target="_blank" title="News Ticker">News Ticker</a>** This news ticker can scroll/slide content in a different style.
* **<a href="https://demo.elementorkit.com/videos-grid/" target="_blank" title="Videos Grid">Videos Grid</a>** Videos Grid with title, content, and details link and self-hosted/ YouTube Videos.

== Installation ==

* Install it as a regular WordPress plugin

* Active the plugin. Make sure you have <a href="https://wordpress.org/plugins/elementor/" rel="nofollow">Elementor</a> plugin installed and active before activating this plugin.

* After installing the plugin follow our step by step <a href="http://docs.wpbean.com/docs/wpb-ea-elementor-addons/">online documentation</a> for using the addons and customize your website.
